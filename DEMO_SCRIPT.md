## 📹 Recording Tips

| Element | Tip |
|---------|-----|
| **Background** | Use OBS with F1 race highlights as subtle background (royalty-free) |
| **Audio** | Use a proper microphone — laptop mics sound unprofessional |
| **Pacing** | Don't rush. Pause after each major point. Let the visuals speak. |
| **Visuals** | Use mouse cursor highlighting (OBS plugin) to show what you're clicking |
| **Lighting** | Well-lit face, no backlight window behind you |
| **Energy** | This is F1 — be excited! But controlled. Think: race engineer energy. |
| **Timing** | Practice 3-4 times before recording. Keep strictly under 3:00. |
